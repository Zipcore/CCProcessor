<h1 align="center">Split color tag</h1>

## Description
Divides a multi-color tag into separate values.

## Example 
\003|\x03|3 - Team color =>

A static sequence of declarations in the config
"{T_Spec}"      "3"     // White
"{T}"           "3"     // Auto
"{T_T}"         "3"     // Red
"{T_CT}"        "3"     // Blue
